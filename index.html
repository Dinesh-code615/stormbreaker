High Tension Transmission Lines Fault Detector
School Project — Real-time Fault Detection for Safer Power Transmission
INTRODUCTION
  • High tension transmission lines are critical for carrying electricity over long distances.
  • Faults such as wire breakage or fire hazards can cause power outages and safety risks.
  • This project aims to detect faults in real time using sensors and alert mechanisms.
SYSTEM OVERVIEW
  • Two endpoints are connected by three copper transmission lines.
  • Each line is monitored by a thermistor sensor.
  • A dual channel relay controls the output indicators (bulbs).
  • The system is powered by a 240V AC supply, stepped down using a voltage regulator.
THERMISTOR SENSOR FUNCTIONS
  • A thermistor is a temperature-sensitive resistor.
  • It detects abnormal heat caused by fire or overload.
  • When a fault occurs, the thermistor changes resistance, triggering the relay.
RELAY OPERATION AND FAULT INDICATION
  • The dual channel relay receives input from the thermistor sensors.
  • If any transmission line is:
    • Overheated (due to fire)
    • Cut or disconnected
  • The relay activates the corresponding bulb to indicate the fault.
  • Each transmission line is linked to a dedicated bulb.
  • When a fault is detected:
    • The bulb turns ON to show which line is affected.
  • This allows for quick identification and response.
WORKING PRINCIPLE SUMMARY
  1. Normal Condition: All lines intact, no bulbs lit.
  2. Fire Fault: Thermistor detects heat ‚Üí Relay activates ‚Üí Bulb lights up.

PROBLEM STATEMENT

High tension transmission lines are prone to fire hazards, insulation failures, and breakages. These faults can cause power blackouts and accidents. Traditional fault detection methods are slow. Hence, an automated fault detection system is required.

LIMITATIONS
  • Thermistors detect only heat, not all faults.
  • Relay-based system is manual, not automated.
  • Only local indication (bulbs) is available, no remote alerts.
FUTURE SCOPE
  • Use of GSM/IoT to send SMS/online alerts.
  • Use of current sensors to detect wire breakage more accurately.
  • Automatic shutdown system to cut faulty line and prevent damage.

DETAILED POWER SUPPLY SECTION

The project is powered from a 240V AC mains supply. This AC voltage is first stepped down using a step-down transformer (230V to 12V). The AC output is then converted to DC using a rectifier, filtered by a capacitor, and regulated to 5V/12V DC using a voltage regulator (such as 7805/7812). This regulated DC powers the thermistor sensors, relay module, and indicator bulbs.

RELAY CLARIFICATION
Relay Requirement:
  • Since the project monitors three transmission lines, each line requires a separate switching mechanism to activate its fault indicator.
  • A 3-channel relay module can be used, where each relay corresponds to one transmission line and its bulb.
  • Alternatively, a combination of one dual channel relay + one single channel relay can also be used to achieve the same function.
  • This ensures that all three lines are independently monitored and faults are clearly indicated without overlap.

SAFETY PRECAUTIONS

Since the project deals with 240V AC and high-tension concepts, safety must be emphasized:

  1. Electrical Isolation
    • Use a step-down transformer to isolate your low-voltage circuit (5V/12V DC) from the dangerous 240V AC mains.
    • This prevents shock hazards.
  2. Fuse Protection
    • Place a fuse on the AC line input to protect against short circuits or overcurrent.
  3. Component Insulation
    • Wires connected to the thermistor (on the transmission lines) should be well insulated to avoid leakage or sparking.
  4. Relay Safety
    • Relays must be rated for the correct voltage/current.
    • Keep high-voltage side (bulbs) separated from the low-voltage control side.
  5. Fire Hazard Prevention
    • Since thermistors detect heat/fire, the project must be tested with safe, low-power heating (like a soldering iron nearby), not with real high-tension lines.
  6. Safe Demonstration
    • For school projects, demonstrate the principle using small bulbs (12V DC) instead of directly working with 240V AC.
